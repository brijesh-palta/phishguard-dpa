# Security and Ethics Scope

This project is an awareness and detection platform. It must not be used for unauthorized phishing, credential harvesting, or external targeting.

## Boundaries
- Use only approved recipients and test accounts.
- Do not collect real passwords, MFA codes, bank details, or secrets.
- Keep landing pages educational and clearly controlled for training.
- Store only the minimum behavior data needed for analytics.
- Use HTTPS and role-based access control in a production deployment.

## Report Language
Describe the simulation module as a controlled security awareness exercise. Emphasize detection, reporting behavior, risk reduction, and employee training outcomes.
